Machine Learning

Machine Learning, and more specifically, Deep Learning, is a field that has exploded in the past decade. The primary goal of machine learning is as follows: Given a dataset, using implicit rules, find a model which accurately describes the data and can make predictions about future data. The applications of the advancements in this field range from Computer Vision to Targeted Advertising to Accurate Medical Diagnoses. Below we define a few key terms used in the machine learning field, as well as provide sample demonstrations of several methods in action.

Training Set

The data set used to train the machine learning model.

Validation Set

The set of points used to test the accuracy of the model inferred from the training set.

Decision Boundary

The hypersurface that partitions the underlying dataset into the respective sub-labels (ie separating leukemia patients of one type from patients of another type).
